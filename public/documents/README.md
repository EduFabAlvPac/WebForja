# Documentos de WebForja

## Metodología FORJA

Esta carpeta debe contener el PDF de la metodología FORJA.

### Archivo requerido:
- `metodologia-forja.pdf` - Documento completo de la metodología FORJA®

### Instrucciones:
1. Coloca tu archivo PDF de la metodología FORJA en esta carpeta
2. Asegúrate de que el archivo se llame exactamente: `metodologia-forja.pdf`
3. El botón de descarga en la sección de metodología enlazará automáticamente a este archivo

### Ruta de acceso:
El archivo será accesible en: `/documents/metodologia-forja.pdf`
